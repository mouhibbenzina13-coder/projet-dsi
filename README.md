# 🎓 EduDSI — Plateforme Étudiante Full-Stack

Application web complète de gestion étudiante 
développée dans le cadre du mini-projet 2ème Année DSI.

## ✨ Fonctionnalités
- 🔐 Authentification JWT
- 📊 Dashboard avec statistiques
- 📚 Gestion des cours
- 📝 Suivi des notes et moyennes
- 📅 Emploi du temps
- 🤖 Agent IA intégré
- 🧠 Quiz de révision

## 🛠️ Technologies
- Frontend  : HTML, CSS, JavaScript
- Backend   : Node.js, Express.js
- Database  : MongoDB, Mongoose
- AI Agent  : Python, Flask
- Auth      : JWT

## 🚀 Installation
# Backend
cd backend && npm install && node server.js

# Agent IA  
cd ai-agent && python agent.py
